# Secure Password Generator

A **Secure Password Generator** built using **Python**, **Flask**, **HTML**, **CSS**, and **JavaScript**. This application allows users to create strong, random passwords with customizable options such as length and inclusion of special characters. The tool ensures your passwords are secure and easy to copy for use.

---

## Features
- 🔒 **Secure Passwords**: Generate strong, random passwords with a mix of letters, numbers, and symbols.
- ⚙️ **Customizable Options**:
  - Set the desired password length (12-64 characters).
  - Toggle inclusion of special characters.
- 🖱️ **User-Friendly UI**:
  - Clean and modern interface with intuitive controls.
  - Copy the generated password to the clipboard with a single click.
- 🌐 **Built with Flask**: Backend powered by Python and Flask for seamless generation and interaction.
- 📱 **Responsive Design**: Works perfectly across desktop, tablet, and mobile devices.

---

## Technologies Used
- **Backend**: Python, Flask
- **Frontend**: HTML, CSS, JavaScript
- **Design**: Responsive layout with modern UI styling

## Installation and Setup

### Prerequisites
- Python 3.12 or higher
- Flask library installed (`pip install flask`)

### Steps
1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/Secured_Password_Generator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Secured_Password_Generator
   ```
3. Install the required Python libraries:
   ```bash
   pip install flask
   ```
4. Run the Flask application:
   ```bash
   python app.py
   ```
5. Open your browser and visit:
   ```
   http://127.0.0.1:5000
   ```

## Usage
1. **Enter Password Length**: Specify the desired password length (default is 16).
2. **Choose Options**: Toggle whether to include special characters in the password.
3. **Generate Password**: Click the "Generate Password" button to create a secure password.
4. **Copy to Clipboard**: Use the "Copy to Clipboard" button to quickly copy the generated password.
